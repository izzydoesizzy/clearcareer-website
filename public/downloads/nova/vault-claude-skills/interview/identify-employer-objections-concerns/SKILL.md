# Identify Employer Objections Concerns

Pinpoint potential employer concerns and develop strategies to counter them.

**Best with:** ChatGPT 3.5, ChatGPT 4.0

**How to use:**
1. Copy the Full Prompt below into ChatGPT
2. Substitute `RED TEXT` with your own details
3. Follow up prompt to produce more specific results

---

## Prompt

Step 1: Copy/Paste the prompt below into ChatGPT, replacing the `RED TEXT` with your details.

```
Based on the job description and my resume attached, make a list of potential objections a hiring manager might have regarding my experience for this role.

Create a 2 column table with a list of the objections and a strategy for how to overcome it.

Objection | Strategy

[ATTACH RESUME FILE & JOB DESCRIPTION FILE]
or PASTE RESUME & JOB DESCRIPTION
```

**Variables to replace:** [ATTACH RESUME FILE & JOB DESCRIPTION FILE]
