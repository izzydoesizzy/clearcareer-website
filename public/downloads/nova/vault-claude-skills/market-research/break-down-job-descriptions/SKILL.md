# Break Down Job Descriptions

Paste a job description and get a clear table of requirements and suggestions

**Best with:** ChatGPT 3.5, ChatGPT 4.0

**How to use:**
1. Copy the Full Prompt below into ChatGPT
2. Substitute `RED TEXT` with your own details

---

## Prompt

Step 1: Copy/Paste the prompt below into ChatGPT, replacing the `RED TEXT` with your details

```
List all the desired skills from the job description below. Create a table with the following 3 columns: 

1. Skill: A brief 2-3 word summary of the skill
2. Required?: Options (Yes, Nice-to-Have)
3. Suggestions: Types of achievements to highlight

Example Output:

**Desired Skills**
Skill | Required? | Suggestions:
Policy Development and Implementation	| Yes | Describe a measurable achievement in developing and implementing privacy policies. What was a measurable outcome?

# Once that table is complete, create another table, that lists any additional skills and experience that would be beneficial to list that are not directly mentioned.

Format the output in a table like the example below:

Example Output:

**Skills Not Directly Mentioned**
Skill | Required? | Suggestions
Stakeholder Management |	Yes	| Describe an achievement where you collaborated across multiple stakeholder groups. What was a measurable outcome?

[COPY/PASTE FULL JOB DESCRIPTION HERE]
```
