# Create Your Skills Section

Quickly create a skills section for your resume

**Best with:** ChatGPT 3.5, ChatGPT 4.0

**How to use:**
1. Copy the Full Prompt below into ChatGPT
2. Substitute `RED TEXT` with your own details
3. Follow up prompt to produce more specific results

---

## Prompt

Step 1: Copy/Paste the prompt below into ChatGPT, replacing the `RED TEXT` with your details.

```
**Prompt:**
Act like a hiring manager and recruitment expert in my field.

First, prompt me to copy/paste my resume, or attach it. 

Then, using the details from my resume, create a new skills section for my resume. There should be 2-3 categories that categorize my skills appropriately. Make sure one of these is tools & software. 

Add an optional certifications Category:

**Category:** Skill1, Skill2, Skill3
**Category:** Skill1, Skill2, Skill3
**Category:** Skill1, Skill2, Skill3
**Certifications:** Cert1, Cert2, Cert3

Last, suggest a comma separated list of skills that aren't on my resume. Ask me to review them to determine if these are skills, tools or software that I have that I simply overlooked, so that they can be added if necessary.
```
